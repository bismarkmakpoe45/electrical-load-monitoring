# electrical-load-monitoring-

Description
A C++ application to track household appliance power consumption and estimate monthly costs.

Features
Save and load appliance data from files.
Calculate daily and monthly kWh usage.
Export a billing summary.
Project Description
This application allows users to monitor electrical load by entering appliance wattage and usage hours. It calculates estimated costs and saves a summary report to a file.

How to Run
Compile using g++ main.cpp -o monitor.
Run the executable ./monitor.
